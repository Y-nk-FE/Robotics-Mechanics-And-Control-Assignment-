## 



机器人学导论课程作业，其中包括了轨迹规划，PD控制器，无外力和有外力的阻抗控制器（无重力补偿）设计和设计报告
规划和设计使用的语言：Matlab
对于需要迁移到嵌入式设备中或者硬件开发者十分友好，都是较为底层的编程语言，可以很快的迁移到java或者C语言控制的控制器
